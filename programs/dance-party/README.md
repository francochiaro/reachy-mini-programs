# Dance Party

Reachy-Mini dances and sings in an infinite loop! Each cycle is ~5 seconds.

## Run

```bash
# Simulator
./run.sh dance-party --sim

# Real robot
./run.sh dance-party
```

Press `Ctrl+C` to stop.

## Dance Moves

- **Head bob** - Up and down to the beat
- **Side to side** - Sway left and right
- **Wiggle** - Quick side shakes
- **Look up/down** - Dramatic head tilt
- **Circle head** - Head moves in a circle
- **Excited shake** - Fast happy shaking

## What It Does

1. Picks a random dance routine
2. Says a random lyric ("La la la!", "Woo hoo!", etc.)
3. Performs 2-3 dance moves
4. Repeats forever until you press Ctrl+C
